# House-Prices-Prediction

В проекте решается задача регрессии — предсказание стоимости недвижимости на основе характеристик объекта с использованием классических методов машинного обучения.

## Постановка задачи
Цель проекта — построить модель, оценивающую цену дома по его характеристикам.  
Такая модель может использоваться для анализа рынка недвижимости, автоматической оценки стоимости объектов и поддержки принятия решений.

## Данные
- Источник: Kaggle — House Prices: Advanced Regression Techniques
- Количество объектов: ~1 460
- Признаки: числовые и категориальные
- Особенности данных: пропуски, выбросы, скошенные распределения, нелинейные зависимости

## Подход
- Разведочный анализ данных (EDA)
- Обработка пропусков с учётом семантики признаков
- Feature engineering и преобразование целевой переменной
- Построение baseline-моделей и моделей градиентного бустинга
- Оценка качества с использованием кросс-валидации
